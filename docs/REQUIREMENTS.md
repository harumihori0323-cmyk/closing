# クロージング支援ツール — 要件定義書

## 1. プロダクト概要

セールスのクロージング商談をリアルタイムでAI支援するWebアプリケーション。
商談前の準備から商談中のリアルタイム応答支援まで、一気通貫でセールス担当者をサポートする。

## 2. ユーザーストーリー

> セールス担当者として、クロージング商談の前にクライアントの状況を整理し、
> 商談中は相手の発言をリアルタイムで書き起こし、AIが最適な返答・反論処理を
> 即座に提示してくれるツールが欲しい。

## 3. 機能要件

### 3.1 顧客情報管理（実装済み）

| 項目 | 状態 | 詳細 |
|------|:----:|------|
| 顧客登録 | 済 | 名前・会社名・業種・役職・提案内容・見積金額・課題・過去のやりとり・備考 |
| 顧客一覧 | 済 | 登録済み顧客のリスト表示 |
| 顧客編集 | 済 | 登録情報の更新 |
| 顧客削除 | 済 | 顧客データの削除 |

### 3.2 クロージング準備（実装済み）

| 項目 | 状態 | 詳細 |
|------|:----:|------|
| 顧客情報サマリー表示 | 済 | 登録情報を一覧表示 |
| AIアドバイス生成 | 済 | 顧客情報に基づき、オープニング・キーポイント・想定反論・クロージングトーク・注意点を生成 |
| アドバイス再生成 | 済 | ワンクリックで再生成可能 |

### 3.3 リアルタイム商談支援（実装済み）

| 項目 | 状態 | 詳細 |
|------|:----:|------|
| 音声認識（書き起こし） | 済 | Web Speech API（ja-JP）で相手の発言をリアルタイム文字起こし |
| 中間テキスト表示 | 済 | 聞き取り中のテキストをリアルタイム表示 |
| AI推奨返答生成 | 済 | 発言確定ごとにAIが推奨返答・意図/戦略・避けるべきことを生成 |
| 会話履歴蓄積 | 済 | 全発言をタイムスタンプ付きで蓄積し、文脈を踏まえた応答を生成 |
| 自動スクロール | 済 | 新しい発言・応答に自動追従 |
| 録音自動再開 | 済 | ブラウザが音声認識を停止した場合に自動再開 |

## 4. 画面構成

| 画面 | パス | 役割 |
|------|------|------|
| 顧客一覧 | `/` | 登録済み顧客の一覧表示・各アクションへの導線 |
| 顧客登録/編集 | `/register`, `/edit/:id` | 顧客情報の入力・編集 |
| クロージング準備 | `/prepare/:id` | AIアドバイス生成・確認 |
| リアルタイム支援 | `/session/:id` | 音声認識 + AI応答のライブセッション |

## 5. 技術スタック

| レイヤー | 技術 |
|---------|------|
| フロントエンド | React 18 + TypeScript + Vite |
| バックエンド | Node.js + Express |
| AI | Claude API（claude-sonnet-4-20250514） |
| 音声認識 | Web Speech API（ブラウザネイティブ） |
| データ保存 | JSONファイル（customers.json） |
| ルーティング | React Router v6 |

## 6. データモデル

### Customer
```typescript
interface Customer {
  id: string;          // タイムスタンプベースのID
  name: string;        // 顧客名
  company: string;     // 会社名
  industry: string;    // 業種
  position: string;    // 役職
  proposal: string;    // 提案内容
  budget: string;      // 見積金額
  challenges: string;  // 顧客の課題・ニーズ
  history: string;     // 過去のやりとり
  notes: string;       // 備考
  createdAt: string;   // 作成日時（ISO 8601）
}
```

## 7. API エンドポイント

| メソッド | パス | 機能 |
|---------|------|------|
| GET | `/api/customers` | 顧客一覧取得 |
| POST | `/api/customers` | 顧客登録 |
| PUT | `/api/customers/:id` | 顧客更新 |
| DELETE | `/api/customers/:id` | 顧客削除 |
| POST | `/api/ai/prepare` | クロージングアドバイス生成 |
| POST | `/api/ai/respond` | リアルタイム応答支援 |

## 8. ユーザーフロー

```
1. 顧客登録
   ユーザーがクライアントの情報（会社名・課題・提案内容等）を入力して保存
       ↓
2. クロージング準備
   「準備」ボタンで遷移 → AIがオープニングトーク・キーポイント・想定反論・
   クロージングトーク・注意点を自動生成
       ↓
3. クロージング開始
   「クロージング開始」ボタンでライブセッションに遷移
       ↓
4. リアルタイム支援
   録音開始 → 相手の発言をリアルタイム書き起こし →
   発言ごとにAIが推奨返答・戦略・避けるべきことを即座に表示
       ↓
5. 商談終了
   「終了」ボタンで一覧に戻る
```

## 10. 動作環境・前提条件

- **ブラウザ**: Google Chrome推奨（Web Speech API対応が必須）
- **マイク**: PCマイクまたはZoom等のシステム音声をキャプチャできる設定
- **API キー**: Anthropic API キーが必要（`.env`に`ANTHROPIC_API_KEY`を設定）
- **Node.js**: v18以上
